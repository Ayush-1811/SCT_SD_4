# E-commerce Product Scraper

This project extracts product information from an online bookstore website and saves the data into a structured CSV file.

## Features
- Scrapes product names
- Extracts product prices
- Captures product ratings
- Exports the result to `products.csv`

## Project Structure
- `main.py` - Python script that performs the scraping
- `requirements.txt` - Required Python packages
- `products.csv` - Output CSV generated after running the script

## Requirements
Install the dependencies using:

```bash
pip install -r requirements.txt
```

## Run the Script
From the project folder, run:

```bash
python main.py
```

## Output
The script will generate a file named `products.csv` containing the scraped data.

## Notes
This scraper is designed for educational use and should comply with the target website's terms of service.
